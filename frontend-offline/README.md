# Bomberman Offline

本地雙人 Bomberman 遊戲，直接在瀏覽器中執行。

使用 **Phaser 3** 製作遊戲畫面，**Vite + TypeScript** 建置。

## 遊戲操作

| 玩家 | 移動 | 放炸彈 |
|------|------|--------|
| 玩家 1 | `W` `A` `S` `D` | `F` |
| 玩家 2 | `K` `M` `,` `.` | `/` |
